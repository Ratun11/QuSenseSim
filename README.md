# QuSenseSim — Training vs Post-Processing Mitigation Patch

This patch adds the missing execution-time mitigation families and makes the mitigation UI easier to understand.

## Added execution-time mitigation families
- DD
- RC
- PT

## Post-processing mitigation families
- ZNE
- PEC
- CDR
- MEM

## UI improvement
The analysis page now separates mitigation techniques into two clearly titled groups:
- Execution-Time Mitigation Techniques
- Post-Processing Mitigation Techniques

## Personalized mitigation
Each sensor can still receive multiple variants, and the personalized selector now makes it clear which family belongs to which mitigation category.
